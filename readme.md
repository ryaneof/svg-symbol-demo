## Requirements

```
Node.js (0.12.*)
```

## Up and Running

```
npm i
npm start
```

### Generate SVG Symbols

SVG source files created by your awesome designer.

```
./svg-source/
```

Default generated file name.

```
./app/images/icon-svg-symbol.svg
```

Generate!

```
node ./generate-svg-symbols.js
```
