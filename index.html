svg-symbol-demo

A simple demo based on svg-sprite, generate SVG symbol.

View on GitHub Download .zip Download .tar.gz

The following four icons request from images/icon-svg-symbol.svg, a combination of these icons using SVG Symbol.

The concept of SVG Symbol is quite simple:
